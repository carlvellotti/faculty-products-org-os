# /customers/interview-notes/ — navigation map

Raw notes from customer calls. The primary source for any quote, complaint, or "what users actually said."

## File naming

`{date}-{account}-{persona}.md`

Examples:
- `2026-04-15-acme-superadmin.md`
- `2026-04-22-northstar-instructor.md`

## Format

Use `/templates/customer-interview.md`. Every interview ends with a `## What we learned` section — 3 bullets max, the most important takeaways.

## When to read what

| You're doing... | Read |
|---|---|
| PRD draft (last 90 days) | All notes for the relevant persona |
| Looking for a specific quote | grep |
| Understanding one account | filter by account slug |

## Cadence

Capture-as-you-go — whoever ran the call drops the notes the same day. If the call has no notes within 24h, it didn't happen as far as the team is concerned.

## Owner

No owner on individual notes — the runner of the call captures. Format/template owner: Carl.
