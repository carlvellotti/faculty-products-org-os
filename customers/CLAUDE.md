# /customers/ — navigation map

Customer-facing context. When you need something here, this is what's where.

## Folders

- `interview-notes/` — raw notes from customer interviews. One file per call: `{date}-{account}-{persona}.md`.
- `account-summaries/` — synthesized account states. One file per account, refreshed quarterly.
- `playbooks/` — how we run customer-facing workflows (interviews, QBRs, churn outreach).

## When to read what

| You're doing... | Read |
|---|---|
| Synthesizing voice-of-customer | `account-summaries/` (skip raw notes unless you need a specific quote) |
| Prepping a customer call | `account-summaries/{account}.md` |
| Looking for a specific quote | `interview-notes/` (search) |
| Running a customer-facing workflow | `playbooks/` |
